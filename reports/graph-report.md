# Graph Report

_**Graphic Report**_ is a report screen consisting of graphs showing the number of runs of the projects within the company in the desired state and their ratios to each other in selected time intervals. When it is first opened, the options are empty.

<figure><img src="../.gitbook/assets/Screenshot 2025-02-20 at 21.27.23.png" alt=""><figcaption></figcaption></figure>

1. Enter the date range.

{% hint style="info" %}
Projects within this selected date range appear in the Unselected Projects field.
{% endhint %}

<figure><img src="../.gitbook/assets/Screenshot 2025-02-20 at 21.28.28.png" alt=""><figcaption></figcaption></figure>

2. Move the projects from Unselected Projects to Selected Projects.
3. Select the status of projects.
4. Click the _**Draw**_ to create the graphics.
5. The screen displays two graphs.

* _**Test Results for Number of Runs**_&#x20;

<figure><img src="../.gitbook/assets/Screenshot 2025-03-13 at 14.21.34.png" alt=""><figcaption></figcaption></figure>

* _**Percentage Representation of Run Results**_

<figure><img src="../.gitbook/assets/Screenshot 2025-03-13 at 14.25.20.png" alt=""><figcaption></figcaption></figure>



* Click on the relevant project column on any chart to view the percentages and execution numbers of plans by project.

{% hint style="info" %}
If the plan to be displayed is only one, it is displayed on the pie chart. For more than one plan, two bar charts showing the number of harnesses and percentages are displayed.

Selected Project can be changed to view the data of another project on this page.
{% endhint %}

<figure><img src="../.gitbook/assets/Screenshot 2025-03-13 at 14.28.47.png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/Screenshot 2025-03-13 at 14.35.52 (1).png" alt=""><figcaption></figcaption></figure>

* Click on the desired plan on the bar chart or pie chart to access detailed scenario-based charts.

{% hint style="info" %}
On this page, the project and its plan can be changed and the graphs of another plan can be displayed.
{% endhint %}

<figure><img src="../.gitbook/assets/Screenshot 2025-03-13 at 14.41.51.png" alt=""><figcaption></figcaption></figure>

* Click on the desired scenario to access the scenario graph page to view the status and running times of the scenarios in detail.
